# Job Application Tracker

A simple CRUD web app for tracking job applications. Users can add, view, update, and delete job entries, including notes and status updates. The dashboard provides a quick overview with status counts, a pie chart, and search functionality.

## Features
- Create, read, update, and delete job applications  
- View full job details including notes  
- Dashboard with status breakdown and Chart.js visualization  
- Search by company or position  
- Clean, minimal interface for easy tracking  

## Tech Stack
- Python (Flask)
- HTML / Jinja2 templates
- Chart.js for dashboard charts

## How to Run
1. Install dependencies  
2. Start the app: python run.py
3. Open your browser and go to: http://localhost:5000

## Purpose
Built to organize and track job applications with a simple, intuitive interface and basic chart.
